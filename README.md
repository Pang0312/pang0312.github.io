# Pang Cheng Siang Portfolio

Upload all files in this folder directly into the `pang0312.github.io` repository root.

Do not upload the folder itself. The repository should show `index.html`, `style.css`, `script.js`, `assets/`, and `projects/` at the top level.

GitHub Pages settings: Settings > Pages > Deploy from a branch > main > /root > Save.

Note: GitHub Pages can run HTML, CSS and JavaScript. PHP/MySQL project features need WampServer locally.

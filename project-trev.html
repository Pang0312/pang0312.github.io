Trev - Asia Trip Planner

Trev is my capstone project. It is an Asia trip planner built using HTML, CSS, JavaScript, PHP, MySQL and WampServer. The project includes user modules, destination browsing, trip planning, reviews, profile pages and admin management functions.

Main Features

Destination browsing, itinerary planning, user login/register, profile management, reviews, admin dashboard, manage users, manage destinations and analytics.

Technologies

HTML, CSS, JavaScript, PHP, MySQL, WampServer, Visual Studio Code.
